# 项目简介

本系统是一款基于Django框架的Python个人记账管理系统，旨在帮助用户方便地记录和管理个人财务收支。

## 核心功能
- 登录注册：支持用户注册和登录，保障账户安全。
- 收入记账管理：用户可记录和查询个人收入信息。
- 支出记账管理：用户可记录和查询个人支出信息。
- 记账数据可视化分析：利用echarts插件，以图表形式展示收入支出情况。
- 最近15天收入与支出分析：查看最近15天的收支详情。
- 近15天的收入支出总数分析：查看近15天收入支出总额。
- 个人信息管理：用户可查看和管理个人资料。

## 适用场景
本系统适用于个人财务管理，帮助用户记录收支情况，进行财务分析。

## 技术特点
- 采用Django框架进行开发，具有良好的可扩展性和稳定性。
- 使用MySQL数据库进行数据存储，保障数据安全。
- 结合echarts插件实现数据可视化分析，提高用户体验。
